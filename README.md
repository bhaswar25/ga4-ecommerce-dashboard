# ga4-ecommerce-dashboard
Interactive ecommerce analytics dashboard using GA4 and Looker Studio

## Project Overview
Built an interactive ecommerce analytics dashboard using GA4 and Looker Studio to analyze:
- Traffic acquisition
- Conversion funnels
- Revenue trends
- Device performance
- Demographic behavior

The dashboard helps identify:
- High-performing traffic channels
- User drop-off points
- Conversion trends
- Revenue-driving customer segments

---

## Tools & Technologies
- Google Analytics 4 (GA4)
- Looker Studio
- Data Visualization
- Ecommerce Analytics
- Marketing Analytics

---

## Key Features
- Executive KPI dashboard
- Traffic source analysis
- Funnel analysis
- Product performance tracking
- Device and geographic analysis
- Interactive filters
